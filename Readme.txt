Instruções:

Para entrar no programa.

1. Para operação do programa, necessário criar o banco de dados do arquivo banco_de_dados.sql;
2. O banco criará junto um usuário elias com senha 'elias' para que o programa acesse o banco;
3. Com o banco de dados criado e aberto, acesse o terminal.
4. Instale o node.js através do site oficial https://nodejs.org/pt/download;
5. No terminal, abra a pasta local com os arquivos descompactados;
6. Digite 'node -v' para consultar a versão do node e se foi instalado corretamente;
7. Instale o pacote de dependências, execute 'npm install' no terminal;
8. Por fim, com o local da pasta aberto no terminal, digite 'node server.js';
9. Abra o navegador e navegue na página: 'http://localhost:3000'

Para visualizar o banco de dados.

1. Acesse o banco de dados pelo PGAdmin 4
2. Em Query Tool, digite 'SELECT * FROM recargas realizadas;'
3. Pressione F5
